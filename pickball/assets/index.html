Android Version

Minimum Android 2.2 version is required for this game.

Pick Ball (V 1.0.2012) Playing Instruction

It is very simple game.The game has no any level, because every time player will play with new level which is generated by game automatic. Player don't know that how many chance will he take to win the game. To play this game, player has to enter any minimum 5 and maximum 7 character string. String contains only a-z,A-Z and 0-9 character. Other characters are not allowed in the game. Click on start button, after clicking start button peacock feather image button will appear. To find match character, player has to click on any peacock feather button. If character matches then it will show in green circle otherwise show in red circle.Game has two empty glasses. One glass is for green and one for red circle. The green or red circle will appear in the glass according to game condition. Player will play the game, while winning message not appear. For this player click on Play again button for next chance.

Menu Section

Game menu has 4 option, which name is Help,New,Stop,History.

  • Help: This option for game help, playing instruction.
  • New: This option for loading new game.
  • Stop: This option for stopping current game.
  • History: This option to view last game history.

Help and History Option View

How to Play

Step:1 Enter any string to game home screen text box, click on start button.

Step:2 Click on any image button which you want to like, after clicking all image button, the play_again button will appear to next round.

Step:3 Repeat round again and again to win game. The progress bar will update in every round.

Step:4After winning the game, the progress bar status has two color (Red and Black). Red status show your final average result. Black status show your complete round progress. To show complete history of last game, to click on menu and four option will show for History click on History button.

Step:5You can view your last game history below by click History button.
